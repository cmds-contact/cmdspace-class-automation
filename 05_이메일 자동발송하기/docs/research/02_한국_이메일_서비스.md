# 한국 이메일 서비스 가이드

> 스티비(Stibee) 중심으로 한국 시장에 최적화된 이메일 서비스 분석

---

## 1. 스티비 (Stibee)

### 개요

스티비는 한국 사용자에게 최적화된 이메일 마케팅 플랫폼입니다. 대부분의 이메일 마케팅 플랫폼이 영어 사이트인 데 비해 **모든 인터페이스가 한글**로 구성되어 있습니다.

### 서비스 특징

| 항목 | 내용 |
|------|------|
| **설립** | 한국 스타트업 |
| **언어** | 완전 한글 지원 |
| **주요 용도** | 뉴스레터, 마케팅 이메일 |
| **2022년 발송량** | 16.2억 건 (전년 대비 1.6배 증가) |

### 가격 정책

| 플랜 | 구독자 수 | 월 비용 | 발송 제한 |
|------|----------|---------|----------|
| 무료 | 500명 | ₩0 | 무제한 |
| 스타터 | 500명 | ₩9,900 | 무제한 |
| 스탠다드 | 3,000명 | ₩33,000 | 무제한 |
| 프로 | 10,000명 | ₩99,000 | 무제한 |
| 엔터프라이즈 | 무제한 | 별도 문의 | 무제한 |

> 발송 건수가 아닌 **구독자 수 기준** 과금

### 주요 기능

#### 이메일 편집기
- 드래그 앤 드롭 편집기
- 다양한 템플릿 제공
- HTML 직접 편집 가능

#### 자동화 기능
- 이메일 통계 기반 자동화 시나리오
- 구독자 정보 기반 트리거
- 설정이 간단하고 직관적

#### A/B 테스트
- 기본 옵션에서 지원
- 제목, 발신자명, 콘텐츠 테스트

#### 통계 분석
- 오픈율, 클릭률 추적
- 구독자별 행동 분석
- 실시간 리포트

### 국내 커머스 연동

스티비는 주요 국내 이커머스 플랫폼과 연동됩니다:

| 플랫폼 | 연동 내용 |
|--------|----------|
| **카페24** | 고객 데이터 자동 동기화 |
| **NHN커머스** | 고객 데이터 연동 |
| **식스샵** | 구독자 연동 |
| **고도몰** | 고객 정보 연동 |

### 자동 관리 기능

- **스팸 처리 자동 삭제**: 스팸 신고한 구독자 자동 제거
- **발송 실패 자동 삭제**: 반송된 이메일 주소 자동 정리
- **구독 취소 처리**: 원클릭 구독 취소 지원

### 2024년 이메일 인증 대응

> **중요**: 2024년 2월부터 Google, Yahoo는 일 5,000건 이상 대량 발송자의 SPF, DKIM, DMARC 미설정 이메일을 스팸 처리합니다.

스티비에서 도메인 레코드 설정 방법:
1. 발신 도메인 설정 메뉴 접속
2. SPF 레코드 추가
3. DKIM 레코드 추가
4. DMARC 정책 설정

## 2. 스티비 vs 해외 서비스 비교

### Mailchimp와 비교

| 항목 | 스티비 | Mailchimp |
|------|--------|-----------|
| **언어** | 한글 | 영어 |
| **가격** | ₩9,900~/월 | $13~/월 |
| **무료 플랜** | 500 구독자 | 500 구독자 |
| **국내 연동** | 카페24, 고도몰 등 | 제한적 |
| **결제 수단** | 원화 결제 | 달러 결제 |
| **고객 지원** | 한국어 | 영어 |
| **A/B 테스트** | 기본 제공 | 유료 플랜 |

### ConvertKit(Kit)과 비교

| 항목 | 스티비 | ConvertKit |
|------|--------|------------|
| **타겟** | 일반 마케터 | 크리에이터 |
| **편집기** | 비주얼 에디터 | 텍스트 중심 |
| **자동화** | 기본 수준 | 고급 수준 |
| **가격** | 저렴 | 상대적 고가 |
| **한글 지원** | ✅ | ❌ |

## 3. 스티비 활용 시나리오

### 시나리오 1: 뉴스레터 운영

```
주간 뉴스레터 발행 워크플로우:
1. 콘텐츠 작성
2. 템플릿 선택/편집
3. A/B 테스트 설정 (제목)
4. 예약 발송
5. 통계 분석
```

### 시나리오 2: 쇼핑몰 마케팅

```
프로모션 이메일 자동화:
1. 카페24/고도몰 고객 데이터 연동
2. 구매 이력 기반 세그먼트 생성
3. 개인화 프로모션 이메일 작성
4. 자동 발송 시나리오 설정
```

### 시나리오 3: 트랜잭션 이메일 (제한적)

> ⚠️ 스티비는 마케팅 중심 서비스로, 트랜잭션 이메일에는 적합하지 않습니다.

트랜잭션 이메일이 필요하면:
- **추천 조합**: AWS SES (트랜잭션) + 스티비 (마케팅)

## 4. 스티비 API

스티비도 API를 제공하여 프로그래밍 방식 연동이 가능합니다.

### API 주요 기능
- 구독자 추가/수정/삭제
- 주소록 관리
- 이메일 발송
- 통계 조회

### 코드 예시 (Python)
```python
import requests

API_KEY = 'your_api_key'
LIST_ID = 'your_list_id'

# 구독자 추가
response = requests.post(
	f'https://api.stibee.com/v1/lists/{LIST_ID}/subscribers',
	headers={
		'AccessToken': API_KEY,
		'Content-Type': 'application/json'
	},
	json={
		'eventOccurredBy': 'MANUAL',
		'confirmEmailYN': 'N',
		'subscribers': [
			{
				'email': 'user@example.com',
				'name': '홍길동'
			}
		]
	}
)
```

## 6. 결론 및 추천

### 스티비를 선택해야 할 때

1. **한글 인터페이스가 중요할 때**
   - 영어에 익숙하지 않은 팀원 포함
   - 한국어 고객 지원 필요

2. **국내 커머스 플랫폼 사용 시**
   - 카페24, 고도몰, 식스샵 등 사용 중
   - 고객 데이터 연동 필요

3. **뉴스레터/마케팅 중심일 때**
   - 정기 뉴스레터 발행
   - 프로모션, 캠페인 이메일

4. **원화 결제를 원할 때**
   - 달러 결제 부담
   - 세금계산서 발급 필요

### 스티비와 다른 서비스 조합

| 용도 | 서비스 조합 | 월 예상 비용 |
|------|------------|-------------|
| 소규모 (마케팅만) | 스티비 무료 | ₩0 |
| 중규모 (마케팅만) | 스티비 스탠다드 | ₩33,000 |
| 트랜잭션 + 마케팅 | AWS SES + 스티비 | ~$1 + 스티비 |
| 대량 발송 | AWS SES 또는 NHN 클라우드 | 종량제 |
