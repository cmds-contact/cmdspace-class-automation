## 04. 결제확인서 생성

**상태**: 🟡 진행중

**설명**: 결제확인서 PDF를 자동 생성하고 저장

**워크플로우**:
```
Airtable → HTML 생성 → PDF 생성 → Cloudflare R2 업로드 → URL을 Airtable에 기록
```

**단계별 작업**:
1. [ ] Airtable에서 정보 가져오기
2. [ ] HTML 생성
3. [ ] PDF 생성
4. [ ] Cloudflare R2 업로드 ← **현재 목표**
5. [ ] R2 URL을 Airtable에 기록

**기술 스택**:
- 데이터: Airtable
- 저장소: Cloudflare R2
